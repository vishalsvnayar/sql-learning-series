# SQL for Data Analysts – 60 Day Learning Series

Welcome to my **60-Day SQL Learning Series** repository.

This repository documents my journey of learning and practicing SQL concepts that are commonly used by **Data Analysts in real-world projects**.

The goal of this series is to help beginners understand SQL step by step through simple explanations and practical examples.

## What You Will Find in This Repository

• Beginner-friendly SQL queries  
• Step-by-step SQL concepts  
• Practical examples used in data analysis  
• Real-world datasets for practice  
• Structured learning from basic to advanced SQL

## Topics Covered

Throughout this 60-day series, I will cover important SQL concepts including:

- SQL Basics (SELECT, WHERE, ORDER BY)
- Aggregate Functions (COUNT, SUM, AVG)
- GROUP BY and HAVING
- SQL Joins (INNER JOIN, LEFT JOIN, RIGHT JOIN)
- Subqueries
- CASE Statements
- Window Functions
- Data Analysis with SQL

## Repository Structure

Each folder contains SQL queries related to specific topics.  
Every file represents a concept explained during the series.

Example structure:

SQL_Basics  
Aggregate_Functions  
SQL_Joins  
Intermediate_SQL  
Advanced_SQL
